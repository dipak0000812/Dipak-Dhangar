# Portfolio Project

A modern, high-performance portfolio website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## üöÄ Features

*   **Responsive Design:** Mobile-first approach offering a seamless experience across all devices.
*   **Performance Optimized:** Dynamic imports, lazy loading, and optimized images using Next.js Image component.
*   **Animations:** Smooth scroll, parity-based list animations, and interactive hover effects powered by Framer Motion.
*   **Typing Effect:** Engaging hero section with dynamic text replacement.
*   **Theme:** Dark mode aesthetic with glassmorphism UI elements.
*   **Data Driven:** Content is separated from logic in the `/data` directory for easy updates.

## üìù Content Management

Update your portfolio data in the `/data` directory:
*   `data/personal.ts`: Basic profile info.
*   `data/experience.ts`: Work history.

*   `data/achievements.ts`: Awards and certs.
